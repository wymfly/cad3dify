# cad3dify 产品化落地路线图与建议

## 1. 产品定位与核心价值
将 `cad3dify` 从一个技术原型（PoC）转型为企业级 3D 打印 SaaS 平台或内部设计中台。核心价值在于：**极低门槛的自然语言/图纸交互 + 工业级的几何生成精度 + 自动化的制造准备。**

## 2. 关键产品化建议

### 2.1 极简双模入口与 3D 实时工作台 (UI/UX)
*   **双模式物理隔离：** 首页提供“机械精密建模（基于图纸/尺寸）”与“自由概念雕塑（基于文本/草图）”两个独立入口，匹配完全不同的用户预期。
*   **WebGL 实时预览：** 集成 `three.js` 或类似引擎，实现模型生成后的 360 度无死角在线预览，提供即时的视觉反馈，这是促成下单的核心。
*   **交互式参数面板：** 对于已生成的机械件，允许用户通过侧边栏滑块（Slider）直接修改关键参数（如孔径、厚度），并实现模型的秒级实时重绘。

### 2.2 自动化工艺拦截与智能报价 (DfAM & Costing)
*   **自动壁厚/悬垂检测：** 后台静默执行 `trimesh` 校验。若壁厚低于打印阈值或存在不可打印的过大悬垂，在 3D 预览中以红色高亮提示，提升良品率。
*   **即时报价系统：** 根据生成的几何体积、包围盒尺寸及选定材料，自动计算耗材成本、机器工时，并给出实时的商业报价与交期预测。

### 2.3 意图确认与品控闭环 (HITL)
*   **中间规格确认页：** 在生成 3D 模型前，向用户展示 AI 提取的结构化参数表（如：“识别到孔数：6，直径：10mm”）。
*   **人工校验确认流：** 强制用户点击“确认尺寸无误”后才进入生产级导出，将 AI 识别幻觉造成的废品风险降至最低，并建立法律免责基础。

### 2.4 资产管理与版本控制 (Persistence)
*   **云端个人/企业零件库：** 记录用户所有的设计历史。支持基于历史记录的“一键克隆”与“微调再生成”，增强客户粘性。
*   **私有模板沉淀：** 允许企业用户将高频生成的特定零件固化为私有参数化模板，建立企业内部的标准件库。

### 2.5 全流程文件导出矩阵 (Export Matrix)
*   **多阶输出：** 
    - 消费级：一键下载 .STL / .3MF（用于家用打印）。
    - 工业级：下载高精度 .STEP / .IGES（用于 CNC 或二次模具设计）。
    - 车间级：直接对接 API 发送给切片服务器。

## 3. 实施优先级建议

1.  **P0 (当前生命线)：** 完善 **HITL 确认流** 与 **SpecCompiler 确定性引擎**。这是解决目前 AI 不稳定、不精确、无法用于生产的最核心环节。
2.  **P1 (商业转化)：** 建立 **3D 在线预览** 与 **自动报价系统**。让用户能看到、敢买。
3.  **P2 (规模化)：** 完善 **有机曲面管道 (方向2)** 与 **个人资产库**。拓展业务边界，建立用户留存。
4.  **P3 (生态位)：** 对接 **工业软件 API (Oqton/Magics)**。实现从设计到制造的全链路全自动化流转。
