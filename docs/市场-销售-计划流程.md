# 市场-销售-计划流程

> [!info] 流程说明
> - 环节==发起人==、==审批人==、==结束人==
> - 环节==触发条件==，如何进入下一步
> - 每个环节==通过==、==不通过==规则
> - **线上环节**、线下环节

---

## 主流程

```mermaid
graph LR
    A[商机] -->|报价需求| B[报价]
    B -->|成单| C[接单]
    C -->|信息推送| D[合同评审]
    D -->|信息推送| E[计划排产]
    E --> F[生产]
    F --> G[售后]

    style A fill:#fff,stroke:#333
    style B fill:#fff,stroke:#333
    style C fill:#fff,stroke:#333
    style D fill:#fff,stroke:#333
    style E fill:#fff,stroke:#333
    style F fill:#fff,stroke:#333
    style G fill:#fff,stroke:#333
```

> [!tip] 信息流标注
> - ==订单、产品信息==：报价 → 接单阶段
> - ==评审信息==：合同评审阶段
> - ==生产信息==：计划排产 → 生产阶段

---

## 各环节详情

### 1. 商机

**操作：商机跟进**

包含信息：
- 商机分类
- 客户名称
- 商机内容
- 分配销售
- 产品模型
- 客户联系方式
- 进度（1F-5F）
- 关联出差

> [!warning] 涉及审批流程
> - 免费订单审批 / 研发费用 / 销售费用
> - 3D打印调试验证申请（走研发费用）
> - 出差审批

---

### 2. 报价

**操作：售前报价**

报价要素：
- 零件信息
- 内部报价单

---

### 3. 接单

**操作：拟定合同或投产单**

包含内容：
- 正式报价单
- 订单号
- 产品类型
- 模型文件
- 技术大纲（是否机加）
- 其他项目文件、资料
- 按不同客户分类

> [!warning] 涉及审批流程
> - 新客户审批
> - 销售订单审批
> - 产品预评审申请

---

### 4. 接单（U9系统）

**操作：U9下订单，接生产系统**

包含信息：
- 客户名称
- 品名 / 品号
- 订货数量
- 价格

> [!abstract] 产品要求评审单
> 接单后生成产品要求评审单，进入合同评审环节。

---

### 5. 合同评审

**操作：线下评审**

包含信息：
- 模型文件
- 材料牌号
- 工艺工序
- 工艺大纲
- 质量检测大纲
- 订单评审状态
- 评审时间
- 项目组 / 负责人
- 预计交期

> [!warning] 涉及审批流程
> - 合同审批
> - 保密协议 / 订单合同

---

### 6. 计划排产

**操作：生产准备**

包含内容：
- 设计后模型
- 设计完成时间
- 设计人员
- 零件高度 / 体积 / 重量
- 切片文件

**操作：生产进度**

包含内容：
- 3D排产
- 根据仓库发货台账
- 生产信息
- 上机时间
- 外协
- 交付清单
- 重点客户进度
- 同步生产实时进度
- 可发货通知

---

### 7. 售后

**操作：U9出货**
- 根据仓库发货台账
- 出货计划

**操作：U9应收单**
- 线下：签收单
- 验收单
- 物流单据
- 开票申请
- 订单合同

> [!warning] 涉及审批流程
> - 成品发货申请（当天发货）
> - 产品紧急放行申请
> - 取消订单审批
> - 特殊开票审批（提前开票）
> - 特殊发货审批（提前发货）

---

## 审批流程汇总

| 阶段 | 审批事项 |
|:-----|:---------|
| 商机 | 免费订单审批、研发费用/销售费用审批、3D打印调试验证申请、出差审批 |
| 接单 | 新客户审批、销售订单审批、产品预评审申请 |
| 合同评审 | 合同审批、保密协议/订单合同 |
| 售后 | 成品发货申请、产品紧急放行申请、取消订单审批、特殊开票审批、特殊发货审批 |
